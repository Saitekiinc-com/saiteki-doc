# 独自モデルの構築

## 背景

組織が拡大する中で最も大きな課題となるのが、**「判断基準の属人化」** と **「質のバラつき」** です。
「誰に聞くかで答えが変わる」「優秀なメンバーのノウハウが退職とともに失われる」──こうした問題を根本的に解決するために、Saitekiではファインチューニング（追加学習）を用いた独自のAIモデルを構築するアプローチを採っています。

## システムの概要

自社データを学習させた独自モデルを構築し、**「SaitekiとしてのS級の基準」** を全社員がいつでも参照できる状態を作ります。

核となるのは **「S級人材の思考プロセス」** を学習データとすることです。単なるマニュアルではなく、優秀な社員が実際にどう考えどう回答したかという「質疑応答（Q&A）」をモデルに学習させます。

- **学習ソース**: S級社員のSlackログ、ドキュメントレビューのコメント、面談記録など
- **目指す姿**: 「もしあの人ならこう答えるだろう」というレベルで、視座の高い回答を返すモデル

構築した独自モデルは、以下の3つの場面で機能します。

### 1. 成果物のレビュー
企画書・設計書・コード・日報などをAIが一次レビューします。Saitekiの品質基準に達しているかを判定するゲートキーパーとして、ベテランが費やしてきたレビュー工数を削減します。

### 2. 悩みの相談・メンタリング
業務上の壁やキャリアの悩みを相談できる相手として機能します。24時間365日利用可能な、S級の判断力を持つメンターとして、手が止まる時間を最小化します。

### 3. 組織状態の可視化
社員とAIの対話ログ自体が、組織の状態を知るための貴重なデータとなります。「このプロジェクトへの技術的な質問が増えている」「ここで判断に迷うメンバーが多い」といった傾向をマネジメント層へ提供します。

## 導入のメリット

1. **判断基準の統一化**: 明文化しきれていない暗黙知や「Saitekiらしさ」をモデルに学習させることで、一貫性のある基準を組織全体に届けられる。
2. **教育コストの削減**: 新入社員がAI壁打ちで提出物の質を引き上げることで、上長の確認・指導コストが減る。
3. **文脈理解による回答精度の向上**: 社内用語や背景を理解した状態で回答が得られ、プロンプトに毎回前提を書く手間が省ける。

## まとめ

独自モデルの構築は、単に「便利なAI」を手に入れることではなく、**トップパフォーマーの思考を組織にスケーリングする**取り組みです。S級人材のナレッジをAIに宿らせることで、組織のどのメンバーでも一定以上の判断品質を担保できる状態を目指します。
