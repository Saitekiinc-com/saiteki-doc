# Level 1: 個人支援 (Individual Support)

## コンセプト
**「個人の作業効率化・スピードアップ」**

AIをあくまで「個人の支援ツール」として活用する段階です。既存の業務フローを大きく変えることなく、個々のエンジニアが手元のタスクをより速く、正確に行うことを目的とします。

## 具体的な取り組み

### 1. コード補完・壁打ち (Copilot / Chat)
- **内容**: GitHub CopilotやCursor、ChatGPT等を活用し、コーディング速度を向上させる。
- **効果**: ボイラープレートの削減、単純ミスの防止、実装ロジックの相談（壁打ち）による手戻り削減。

### 2. 調査支援 (AI Search)
- **内容**: Perplexityやブラウザ内AI検索を活用し、技術調査やライブラリ選定を効率化する。
- **効果**: 従来のGoogle検索よりも素早く、文脈に沿った回答を得ることで、調査時間を短縮する。

### 3. パイプラインへの組み込み (Process Automation)
- **内容**: 開発パイプラインの一部にAIツールを組み込む。
    - **CodeRabbit**: Pull Requestに対するAI自動レビュー導入し、単純な指摘やTypo修正を自動化する。
- **効果**: レビュワーの負担軽減、レビューサイクルの高速化。

### 4. 要件定義の効率化 (Requirement Definition)
- **内容**: 曖昧な要望やメモ書きから、AIを活用して要件定義書のドラフト版を生成する。
- **効果**: ドキュメント作成の初動スピード向上、抜け漏れの防止。
